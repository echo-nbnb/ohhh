# 字体下载说明

## 推荐字体

### 1. 思源黑体 (Source Han Sans)
- **用途**：标题、正文
- **风格**：现代黑体，清晰易读
- **下载**：https://github.com/adobe-fonts/source-han-sans/releases
- **选择文件**：SourceHanSansSC.zip 中的 OTF/Genkai 后缀

### 2. 思源宋体 (Source Han Serif)
- **用途**：引用、装饰性文字
- **风格**：衬线体，古典优雅
- **下载**：https://github.com/adobe-fonts/source-han-serif/releases
- **选择文件**：SourceHanSerifSC.zip

### 3. 阿里巴巴普惠体
- **用途**：正文
- **风格**：现代简洁
- **下载**：https://alibabafood.github.io/font/

## 放置位置

将下载的字体文件放入 `assets/fonts/` 目录

## 使用说明

代码会自动查找以下路径：
1. `assets/fonts/{fontname}.ttf`
2. Windows系统字体 `C:\Windows\Fonts\{fontname}.ttf`
